### Description

zabbx agent (active) 模是由agent端自己主动去找服务器，包括在注册到zabbix Server端时也是agent端主动注册上去的，监控时使用active模式，则是agent端主动给予server端监控数据，agent端主动发起，这种情况下可以节省server端的性能。
